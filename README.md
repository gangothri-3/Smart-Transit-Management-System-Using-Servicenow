The **Smart Transit Management System** is a ServiceNow-based application designed to digitize and automate public bus transportation operations. It provides an end-to-end solution for **ticket booking, route and fleet management, payments, reporting, and passenger support** through a centralized self-service portal.

The system leverages **ServiceNow platform capabilities** such as Catalog Items, Flow Designer, Business Rules, ACLs, Reports, Dashboards, and Knowledge Base to improve efficiency, transparency, and user experience for both passengers and administrators.
